# PPE Violation Detection

This is an ML project that detects PPE (Personal Protective Equipment) violations on construction sites. The goal is to identify if someone is not wearing their safety kit properly.

## Setup Instructions

To run the project, follow these steps:

1. Download all files from the Google Drive link provided.
2. Extract the zip file.
3. Open the terminal and navigate to the project folder.
4. Inside the folder, create a virtual environment.
